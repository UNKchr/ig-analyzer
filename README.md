# Instagram Follower Analyzer

Tampermonkey script to analyze followers vs following on Instagram.

## Features

- Detect users who don't follow back
- Track changes between executions
- Persistent snapshots
- Draggable UI panel

## Installation

1. Install Tampermonkey
2. Open the script:
   <https://raw.githubusercontent.com/UNKchr/ig-analyzer/main/instagram-follower-analyzer.user.js>
3. Click Install

## Usage

Open Instagram, click "Run Analysis" in the panel.
